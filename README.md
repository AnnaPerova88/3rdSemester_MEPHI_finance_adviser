# 3rdSemester_MEPHI_finance_adviser

"Прогнозирование временных рядов с Telegram-ботом"
О проекте
Telegram-бот для анализа и прогнозирования временных рядов финансовых данных. Проект включает в себя:

ML-модели для прогнозирования (ARIMA, Ridge Regression, LSTM)
Визуализацию исторических данных и прогнозов
Пользовательский интерфейс через Telegram

Ключевые особенности
Множество моделей: Реализованы и сравнены разные подходы к прогнозированию

Рабочие модели: Все модели функционируют и дают прогнозы
Telegram-интерфейс: Бот с кнопками для удобного взаимодействия
Исторические данные: Возможность загрузки и отображения исторических данных через бота

Структура проекта
ML-модели в Colab:

ARIMA, Ridge Regression, LSTM
Сравнение моделей по метрикам RMSE и MAPE

Визуализация прогнозов
Telegram-бот:
Интерфейс с кнопками

Загрузка исторических данных
Базовый функционал для взаимодействия
